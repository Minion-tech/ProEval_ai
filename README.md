# ProEval AI

ProEval AI is an advanced AI-powered platform designed for evaluating student projects and automating technical vivas.

## Architecture
- **Frontend:** Next.js (TypeScript)
- **Backend:** FastAPI (Python)
- **Database:** PostgreSQL
- **AI Agents:** ElevenLabs Conversational AI, Google Gemini, Groq, and NVIDIA NIM.
